This is a modified version of the standard Node.js chat room. Both the client-side and server-side are written in javascript.

It has been modified to function as both a FeedHenry Mobile App and a Web App.

The FeedHenry Mobile App is communicating with the cloud code via the $fh.act() FeedHenry API.
